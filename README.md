# 🏭 Smart Factory Safety System

An AI-based intelligent safety monitoring system for factories using **Computer Vision** to detect and prevent workplace accidents in real time.

---

## 🎯 Overview

This system uses **multiple AI models** to monitor factory workers through live cameras and detect:
- 🪖 PPE compliance (Helmet & Vest detection)
- 🤕 Fall detection
- ⚠️ Collision or proximity with forklifts/machines
- 🧍‍♂️ Abnormal posture or prolonged lying
